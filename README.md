# Opis Projektu
BudgetBlue to nowoczesne narzędzie CLI (Command Line Interface) do zarządzania finansami osobistymi. Aplikacja została zaprojektowana z myślą o szybkości działania i estetyce, wykorzystując bibliotekę `Rich` do generowania interfejsu TUI (Text User Interface). Projekt realizuje temat "Narzędzie CLI" zgodnie z wymaganiami przedmiotu "Techniki Efektywnego Programowania".

# Funkcjonalności
* **Dodawanie transakcji:** Obsługa przychodów i wydatków.
* **Kategoryzacja:** Przypisywanie opisów i kategorii do operacji.
* **Historia operacji:** Przejrzysta tabela z kolorowaniem składni (czerwony dla wydatków, zielony dla przychodów).
* **Persystencja danych:** Zapisywanie stanu w pliku JSON.
* **Testy:** Weryfikacja logiki biznesowej za pomocą `unittest`.

# Technologie
* **Język:** Python 3.8+
* **UI Framework:** Rich, Typer
* **Testy:** Unittest
* **Kontrola wersji:** Git & GitHub
